32位系统使用Windows 7 USB DVD Download Tool创建64位系统安装U盘

在32位Windows系统电脑上，使用 Windows 7 USB DVD Download Tool 创建64位Windows（Win7/8/2008/2012）系统安装U盘时最后会出现错误提示，导致安装U盘内引导记录没有正确写入，使用此U盘无法正常启动安装系统。 具体错误信息： Files copied successfully. However, we were unable to run bootsect to make the usb device bootable.

准备升级NAS

一年前DIY的NAS，24小时挂PT，稳定运行中...
抵制不住ZFS的诱惑，决心升级硬件，搭建一台FreeBSD ZFS NAS/Home Server。

目前的配置：

  • CPU Intel Pentium G630T
  • 主板 BioStar TH61 ITX
  • 内存 Apacer DDR3 1333 2G
  • 机箱 CFI A7879 Mini-ITX Case

这个机箱真心不错，我买的是老版的，据说新版的可以多装一个额外的2.5寸HDD

  • 硬盘 Seagate Barracuda Green ST2000DL003×3

系统和软件：

  • Ubuntu Server 12.04.2 LTS
  • Linux MD RAID5

网上说RAID5有Write Hole的严重问题O_o，linux MD RAID5算法应该是相同的，同样会有这个问题存在。我可不想让多年的珍贵照片和重要文档就这样消失，电影倒是可以重新下载...这是此次升级的最主要原因!

  • rutorrent /rTorrent 挂PT
  • Samba 文件共享
  • Twonky Server 做DLNA服务器
  • Xeoma 做视频监控服务器（目前就一个IP Camara......）

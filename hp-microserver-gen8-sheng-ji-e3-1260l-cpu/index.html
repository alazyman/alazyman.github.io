HP MicroServer Gen8 CPU升级 E3-1260L

我的 HP MicroServer Gen8 原配的CPU为 G2020T。

后来升级了低功耗神器 E3-1220L V2。

E3-1220L V2双核4线程，总感觉性能偏弱，特别是在作为虚拟主机的情况下。

最近把 Gen8 从 Esxi 迁移到了 Hyper-V，于是动起了升级CPU的念头。

其实心中早就对 E3-1265L V2 长了草，无奈实在太贵了。

逛某宝时看见 E3-1260L 的价格不错。

其实E3-1260L和 E3-1265L V2 差得不多，抛开平台、制程什么的不谈，也就是基础频率低了0.1GHz，最大睿频低了0.2GHz，最大内存频率1333和1600，还有就是PCIe 2.0和3.0的区别......

但是 E3-1265L V2 价格就翻了一倍还多。

果断入手 E3-1260L 。

装机....
E3-1260L

换下的E3-1220L V2

POST..... Gen8 很好的识别新的CPU

来看看iLo4的信息

CPU

内存，可以看见1600MHz的内存运行在1333MHz

固件，系统Rom已经升到新版的。

进Windows Server 2016，开了4个虚拟机，看看CPU温度。

40度左右，高是不高。但是....Tj.Max只有73度，也就是说到了这个温度的就会自动降频保护，73是个很低的值，一般Intel的好像都在100度....

最高达到70度，是我短时间烤机时发生的，看来得琢磨这CPU用主动散热方式了，否则被动散热器在CPU大负荷时有点抗不住，当然一般情况是绝对没有问题的。

再看看ilo里温度传感器和风扇转速的信息，一般情况下很好。

继续观察吧。

NexentaStor下获取真正的root权限

NexentaStor 是 Solaris 开源分支的一个封装发行版本。

默认的控制台是NMC shell，如果要进入UNIX bash，可以这么做：

用root账号登录，然后.....

nmc@homenas:/$ option expert_mode = 1
nmc@homenas:/$ !bash
You are about to enter the Unix ("raw") shell and execute low-level Unix command(s). Warning: using low-level Unix commands is not recommended! Execute?  (y/n)

按"y"后，就进入bash了！

nmc@homenas:/volumes#

退出bash，

nmc@homenas:/volumes# exit
exit

系统提示：

Important: To re-sync the appliance's management state information, please
consider running 'setup appliance nms restart' command.

nmc@homenas:/$

为系统完整性考虑，你应该考虑运行它建议的命令”setup appliance nms restart”。

Solaris 11.2 下禁用 LSI 9211-8i SAS HBA卡的 SATA NCQ
